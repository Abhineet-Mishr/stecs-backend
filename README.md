STeCS
Smart Triage and Emergency Coordination System

STeCS is an AI-powered platform designed to support frontline healthcare workers in emergency patient assessment, triage, and referral coordination. 
The system aims to improve early recognition of critical illness, standardize triage workflows, and facilitate efficient communication between healthcare facilities, particularly in resource-constrained settings.

Features:-
AI-assisted patient triage
Clinical decision support
Emergency Severity Index (ESI) integration
Resource-aware referral recommendations
Real-time coordination dashboard
Offline-first architecture
Multilingual and voice-guided workflows
